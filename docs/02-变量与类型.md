# 变量与类型

## 基本类型

TransPyC 支持 Python 内置类型和 `t` 模块的扩展 C 类型。

### 内置类型

```python
a: int = 42                  # int
b: float = 3.14              # float
c: str = "hello"             # char*
d: bool = True               # bool
```

生成 C：

```c
int a = 42;
float b = 3.14;
char* c = "hello";
bool d = 1;
```

### t 模块类型

```python
import t

x: t.CChar = 'A'             # char
y: t.CShort = 100            # short
z: t.CLong = 1000000L        # long
f: t.CFloat = 3.14f          # float
d: t.CDouble = 3.14159265    # double
```

生成 C：

```c
char x = 'A';
short y = 100;
long z = 1000000L;
float f = 3.14f;
double d = 3.14159265;
```

### 带初始化

```python
count: int = 0
name: str = "TransPyC"
pi: float = 3.14159
```

### 仅声明（不初始化）

在初始化表达式中写 `c.State` 表示"仅声明，不赋初值"：

```python
import c

x: int = c.State              # int x;
ptr: str = c.State            # char* ptr;
```

生成 C：

```c
int x;
char* ptr;
```

> 翻译器识别 `= c.State` 后省略 `= 值` 部分，只生成声明语句。

## 存储修饰符

### static

```python
import t

counter: t.CStatic | int = 0      # static int
```

生成 C：

```c
static int counter = 0;
```

### extern

```python
import t

external_var: t.CExtern | int     # extern int
```

### const / volatile

```python
import t

MAX: t.CConst | int = 100         # const int
flag: t.CVolatile | int = 0       # volatile int
```

## 指针类型

```python
import t

# 整数指针
p: t.CPtr | int                    # int*
buf: t.CPtr | t.CChar              # char*

# 结构体指针
task: t.CPtr | t.CStruct(name="TASK")  # struct TASK*
```

生成 C：

```c
int* p;
char* buf;
struct TASK* task;
```

## 数组类型

### 一维数组

```python
import t

arr: t.CChar[16]                   # char[16]
nums: int = [1, 2, 3, 4]           # int nums[] = {1, 2, 3, 4}
```

生成 C：

```c
char arr[16];
int nums[] = { 1, 2, 3, 4 };
```

### 数组访问

```python
arr[0] = 10
x = arr[i]
```

生成 C：

```c
arr[0] = 10;
x = arr[i];
```

## 类型转换

### c.TypeCast

```python
import c

x: int = 42
y = c.TypeCast('float', x)         # ((float)x)
```

生成 C：

```c
int x = 42;
int y = ((float)x);
```

### t 模块类型转换

```python
import t

val = t.CInt(some_expr)            # ((int)some_expr)
ptr = t.CType(addr, MY_STRUCT, t.CPtr)  # ((struct MY_STRUCT *)addr)
```

## sizeof

```python
# 数组长度
arr: int = [1, 2, 3]
n = len(arr)                    # sizeof(arr) / sizeof(arr[0])
size = sizeof(int)              # sizeof(int)
```

## 类型组合规则

`|` 运算符按以下优先级合并 C 类型名：

| 优先级 | 规则 | 示例 |
|--------|------|------|
| 1 | 存储修饰符优先 | `static \| int` → `static int` |
| 2 | 指针置后 | `int \| *` → `int*` |
| 3 | 类型合并 | `long \| int` → `long int` |
| 4 | struct 优先 | `struct \| *` → `struct *` |

```python
import t

# static int
counter: t.CStatic | int

# struct TASK*
task: t.CPtr | t.CStruct(name="TASK")

# const char*
msg: t.CConst | t.CPtr | t.CChar
```
